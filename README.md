# HOWTO
本repo内source.html提供的是静态js调用模式，将其加入您的网页模板，并将chtholly文件夹置于网站根目录下。

您也可以通过向您的博客中加入插件，并将其原模型替换为本repo内的lpk文件来实现此功能。
